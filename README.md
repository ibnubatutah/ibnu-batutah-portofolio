Perkenalkan nama saya Ibnu Batutah dan selamat datang di website portofolio saya! Website ini adalah tempat saya memamerkan hasil karya dan pengalaman di dunia pengembangan web.

## Teknologi & Keahlian Teknis

### Front-End Development
*   **Core:** HTML5 & CSS3 (Dekstop & Responsive Design)
*   **JavaScript:** Vanilla JS, **Vue.js**, dan **React.js**
*   **Optimization:** Debugging performa website menggunakan browser Devtools (seperti optimasi resource `link rel` untuk mencegah pemborosan bandwidth akibat kesalahan penggunaan `preload`).

### Back-End & Infrastructure
*   **Language:** **PHP** (Server-side scripting)
*   **Cloud Hosting:** Pengelolaan hosting pada platform **Hostinger** & **Rumahweb**.

### Web Troubleshooting & Maintenance
*   **Security (SSL):** Berpengalaman menangani masalah **SSL Expired** dan melakukan reaktivasi melalui fitur **Auto-run SSL** di cPanel/hPanel untuk memastikan website tetap aman (HTTPS).
*   **Performance Debugging:** Mahir menggunakan **Browser DevTools** untuk mengidentifikasi dan memperbaiki error pada sisi kodingan maupun jaringan.
